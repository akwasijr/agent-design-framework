# Agent Design Framework

## From UX to AX: A Thinking Framework for Agent-Managed Work

Work that has traditionally been done by people is increasingly handled by AI agents: software that can interpret goals, make decisions, take actions, and learn from outcomes. This shift is already underway across industries, from customer service to financial analysis to healthcare coordination.

When an agent sits between the human and the outcome, the design questions change. How do humans delegate to agents, supervise them, and course-correct when things go wrong? How is trust built? How is control maintained?

This framework addresses those questions. The design object is no longer the screen. It's the **relationship** between human and agent.
