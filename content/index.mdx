# Agent Design Framework

## From UX to AX: A Thinking Framework for Agent-Managed Work

The AI landscape is changing fast. Work that has traditionally been done by people is increasingly being handled by AI agents: software that can interpret goals, make decisions, take actions, and learn from outcomes. This is not a distant future. It is happening now across industries, from customer service to financial analysis to healthcare coordination.

If agents are becoming the key actors in how work gets done, then the way we design experiences needs a fundamental overhaul. Traditional user experience design assumes a human operating an interface directly: clicking, scrolling, filling in forms. But when an agent sits between the human and the outcome, the questions change entirely. How do agents work? How do humans work with them? How do we interact with agents, delegate to them, supervise them, and course-correct when things go wrong?

This framework exists to think through those questions. It is not about making better chatbots or smarter assistants. It is about designing the relationship between humans and autonomous systems: how trust is built, how control is maintained, how transparency is provided, and how we ensure that agents act in the interest of the people they serve.

The design object is no longer the screen - it's the **relationship** between human and agent.
